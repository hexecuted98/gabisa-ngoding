Gimana Sih Cara Kerjanya?

Oke, sekarang kita bahas dengan serius tapi santai. Jadi bayangin lo punya otak berupa array memori isinya 30.000 slot kosong, semua diisi angka nol.

Terus lo punya satu pointer, alias jarum yang nunjuk ke salah satu slot itu. Nah, setiap perintah di Gabisa Ngoding (alias di Brainfuck) itu mainin jarum dan angka-angka di memori itu.

Skema Kerja Otaknya

  1. Program dipecah jadi token (kayak kata puh, gw, dsb)
  2. Token dikompilasi jadi bytecode (angka-angka simpel)
  3. Bytecode dijalankan oleh mesin virtual (VM) yang nyimpen data di array
  4. Setiap perintah ubah pointer, isi memori, atau print karakter
