> Bahasa Gabisa-Ngoding terinspirasi dari Brainfuck, yaitu salah satu bahasa pemrograman esoterik paling terkenal di dunia yang hanya memiliki 8 simbol:

| Brainfuck | Fungsi                               |
| --------- | ------------------------------------ |
| `>`       | Geser pointer ke kanan               |

| `-`       | Kurangi nilai pada sel               |
| `.`       | Cetak karakter dari sel              |
| `,`       | Baca karakter ke sel                 |
| `[`       | Jika nilai 0, lompat ke setelah `]`  |
| `]`       | Jika tidak 0, kembali ke setelah `[` |
